# Dominator

If you love [PSYCHO-PASS](https://psycho-pass.com), this theme is the best for you.

## Installation

Launch Quick Open with:

- Linux `ctrl+P`
- macOS `cmd+P`
- Windows `ctrl+P`
```
ext install dominator
```
And pick the one by tomocy as author.
